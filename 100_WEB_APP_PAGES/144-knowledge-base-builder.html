Knowledge Base Builder - UI Flow

State 1: Home Page (Logged Out)

State 2: About Page

State 3: Sign Up Page

State 4: Sign Up Page (Validation Error)

State 5: Login Page

State 6: Login Page (Authentication Error)

State 7: Forgot Password Page

State 8: Password Reset Page (Using Token Link)

State 9: Dashboard / My Knowledge Bases (Logged In)

State 10: Create New Knowledge Base Page

State 11: Create New Knowledge Base Page (Validation Error)

State 12: Knowledge Base View (Editor - Empty State)

State 13: Knowledge Base View (Editor - With Content)

State 14: Create/Edit Article Page

State 15: Create/Edit Article Page (Validation Error)

State 16: Public Article View

State 17: Search Results Page (Public View)

State 18: Account Settings Page (Logged In)
