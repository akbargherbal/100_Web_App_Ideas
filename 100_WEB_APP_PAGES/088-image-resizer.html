Image Resizer - UI Flow

This document outlines the user interface flow for the Image Resizer web application using static HTML and Bootstrap.

State 1: Home Page (Upload)

State 2: Image Uploaded & Configuration

State 3: Processing

State 4: Result/Download

State 5: About Page

State 6: Home Page - Upload Error (Invalid File Type)

State 7: Configuration - Input Error (Invalid Dimensions)
