Interactive Map Route Planner - UI Flow

State 1: Home Page

State 2: About Page

State 3: Login Page

State 4: Signup Page

State 5: Route Planner (Initial State)

State 6: Route Planner (Drawing Route)

State 7: Route Planner (Route Drawn / Finished)

State 8: Route Planner (Saving Route - Modal)

This state shows the main planner view with the "Save Route" modal dialogue open.

State 9: Route Planner (Saving Route - Loading)

Shows the modal with a loading state after clicking "Save Route".

State 10: Route Planner (Saving Route - Success)

Shows the modal with a success message after saving.

State 11: Route Planner (Saving Route - Error)

Shows the modal with an error message if saving fails.

State 12: Route Planner (Exporting GPX - Feedback)

Shows feedback after clicking "Export GPX". Actual download is handled by the browser, so this is just visual confirmation/initiation.

State 13: My Routes Page (List View)

State 14: My Routes Page (Loading)

Shows placeholder content while routes are being loaded.

State 15: My Routes Page (Empty State)

State 16: View Route Details

Shows a read-only view of a specific saved route, likely accessed from "My Routes".

State 17: Error Page (Generic/404)
