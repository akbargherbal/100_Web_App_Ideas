Workout Logger - UI Flow

State 1: Home Page (Logged Out)

State 2: Sign Up Page

State 3: Sign Up Page (Validation Error)

State 4: Login Page

State 5: Login Page (Error)

State 6: Home Page (Logged In) / Dashboard

State 7: Start/Log New Workout Page

State 8: Add Exercise Modal (Static View)

State 9: Add Exercise Modal (Validation Error)

State 10: Workout Logged Confirmation (Feedback)

State 11: Workout History Page

State 12: View Specific Workout Detail

State 13: Exercise List/Database Page

State 14: User Profile Page

State 15: About Page
