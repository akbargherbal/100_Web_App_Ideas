Time Tracking App - UI Flow

State 1: Home Page (Logged Out)

State 2: About Page

State 3: Sign Up Page

State 4: Sign Up Page (Validation Error)

State 5: Login Page

State 6: Login Page (Auth Error)

State 7: Dashboard (Logged In - Timer Stopped)

State 8: Dashboard (Logged In - Timer Running)

State 9: Projects Page (List)

State 10: Add Project Page

State 11: Add Project Page (Validation Error)

State 12: Time Entries/Log Page (List)

State 13: Manual Time Entry Page

State 14: Manual Time Entry Page (Validation Error)

State 15: Reports Page

State 16: Account Settings Page
